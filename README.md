LIBRARY MANAGER

Описание проекта

Library Manager — консольное приложение на языке программирования C++, предназначенное для управления библиотекой. Программа позволяет хранить информацию о книгах и журналах, выполнять поиск, удаление и просмотр данных.

Проект разработан с использованием принципов объектно-ориентированного программирования (ООП).

---

Функциональные возможности

1. Добавление книги.
2. Добавление журнала.
3. Просмотр списка всех изданий.
4. Удаление издания по ID.
5. Поиск издания по названию.
6. Просмотр статистики библиотеки.

---

Используемые технологии

Язык программирования: C++

Среда разработки: Microsoft Visual Studio 2022

Тип приложения: Консольное приложение

---

Реализованные принципы ООП

1. Инкапсуляция

Данные объектов хранятся внутри классов и доступны через методы класса.

2. Наследование

От базового класса BookItem наследуются классы:

- Book
- Magazine

3. Полиморфизм

Метод display() объявлен виртуальным в базовом классе и переопределён в классах-наследниках.

4. Абстракция

Класс BookItem является абстрактным и содержит общие характеристики библиотечных изданий.

---

Структура проекта

BookItem.h / BookItem.cpp

Базовый абстрактный класс, содержащий:

- ID издания;
- название;
- год выпуска.

Book.h / Book.cpp

Класс книги, дополнительно содержащий:

- автора.

Magazine.h / Magazine.cpp

Класс журнала, дополнительно содержащий:

- номер выпуска.

LibraryManager.cpp

Главный файл программы, содержащий меню и основную логику работы приложения.

---

Статистика

Программа позволяет получить следующую статистику:

- количество книг;
- количество журналов;
- общее количество изданий в библиотеке.

---

Заключение

В ходе выполнения проекта были изучены и применены основные принципы объектно-ориентированного программирования: инкапсуляция, наследование, полиморфизм и абстракция.

Программа демонстрирует работу с классами, виртуальными функциями, контейнером vector, динамической памятью и консольным пользовательским интерфейсом.
